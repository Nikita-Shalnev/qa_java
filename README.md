
QA Java Project,Sprint 6
Этот проект представляет собой набор автоматизированных JUnit-тестов, написанных на Java с использованием Maven. Основная задача — протестировать часть приложения, в которое записываются наблюдения за животными.


Технологии
Java 11, JUnit 4.13.2, Mockito 5.21.0

Настройка среды
Клонировать репозиторий: git@github.com:Nikita-Shalnev/qa_java.git Импортировать проект в IntelliJ IDEA. Убедиться, что установлен JDK 11 Maven подтянет зависимости автоматически при открытии проекта.
